# Simple Todo App

A lightweight and user-friendly Todo application that helps users keep track of tasks. With this app, you can create todo items, mark them as completed, and remove them once they’re no longer needed. Perfect for personal task management or daily planning.

## Functionality

Add new tasks – Enter a task name and optional due date to keep your list organized.

Mark tasks as completed – Easily check off tasks as you complete them.

Delete tasks – Remove tasks that are finished or no longer relevant.

Form validation – Prevents submission of empty tasks and disables the submit button until the input is valid.

Persistent UI features – Each task has unique IDs for proper handling and interactions.

The app uses a modern JavaScript class-based approach for both tasks (Todo) and form validation (FormValidator), making the code clean and modular.

## Technology

HTML5 – Structure of the page, including templates for todo items and form elements.

CSS3 – Basic styling for the todo list, buttons, and popup forms.

JavaScript (ES6+) – Functionalities like creating todos, checking them off, deleting them, and form validation.

UUID library – Generates unique IDs for todo items to ensure proper DOM management.

DOM manipulation – Cloning templates, adding event listeners, and updating the UI dynamically.
